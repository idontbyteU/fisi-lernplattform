Algorithmus zum Zusehen

Bubblesort animiert

Bubblesort vergleicht immer zwei Nachbarn und tauscht sie, wenn sie falsch herum stehen. Große Werte „blubbern“ nach rechts. Gelb = wird verglichen, Rot = wird getauscht, Grün = fertig sortiert.

📊 Sortier-Visualizer
Drücke „Sortieren“ – oder geh „Schritt“ für Schritt durch.
def bubblesort(liste): n = len(liste) for i in range(n - 1): for j in range(n - 1 - i): if liste[j] > liste[j + 1]: liste[j], liste[j + 1] = liste[j + 1], liste[j] return liste
Warum lernen? Bubblesort ist langsam, aber perfekt zum Verstehen, wie ein Algorithmus überhaupt arbeitet. Im Alltag nimmt man liste.sort() – das ist hochoptimiert.

Dein Lernpfad

Python von null an, jeder Schritt baut auf dem vorherigen auf. Lies die Idee, lade das Beispiel ins Code-Labor und probiere es aus. Kein Schritt setzt etwas voraus, das du noch nicht kennst.

1 · Variablen & Datentypen

Eine Variable ist ein benannter Behälter für einen Wert. Python merkt sich den Typ automatisch.

  • int – ganze Zahl: alter = 20
  • float – Kommazahl: groesse = 1.80
  • str – Text (in Anführungszeichen): name = "Max"
  • bool – Wahrheitswert: aktiv = True
Im Code-Labor zeigt der Variablen-Inspektor jeden dieser Werte mit Typ an.

2 · Rechnen & Operatoren

Die Grundrechenarten plus drei, die oft in Prüfungen vorkommen:

  • ** Potenz · % Modulo (Rest) · // Ganzzahldivision
  • Beispiel: 10 % 3 ergibt 1 (Rest), 10 // 3 ergibt 3

3 · Ein- und Ausgabe

print(...) gibt aus, input(...) fragt ab. Wichtig: input() liefert immer Text – zum Rechnen erst mit int() oder float() umwandeln.

4 · Entscheidungen (if / elif / else)

Das Programm wählt einen Weg je nach Bedingung. Python prüft von oben nach unten – die erste passende Bedingung gewinnt.

5 · Schleifen (for & while)

Wiederholungen. for läuft über einen festen Bereich, while solange eine Bedingung wahr ist. → Sieh dir das im Tab Schleifen an.

6 · Listen

Mehrere Werte in einer geordneten Sammlung. → Tab Listen zeigt jede Operation als Bewegung.

7 · Zufall & Statistik (dein Unterrichtsstoff)

Mit random Zufallszahlen erzeugen, mit Listen sammeln und auswerten – genau die Würfel-Aufgabe aus der Schule. Lade sie im Code-Labor unter „Würfel-Statistik“.

8 · Eigene Funktionen (def)

Code, den du benennst und wiederverwendest. def name(parameter): … return wert. Damit baust du größere Programme aus kleinen Bausteinen.
